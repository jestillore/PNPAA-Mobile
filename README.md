PNPAA-Mobile
============

PNPAA basic collaboration tool.

Features
---------
- Forum (categorized by department)
- Officers and Members Database
- Events

Technologies
------------
- [mobile-angular-ui](http://mobileangularui.com/docs/)
- [Cordova](http://cordova.apache.org/)
